# 🏋️ Fitness Tracker AI

An AI-powered fitness analytics and workout recommendation platform built using React, Tailwind CSS, Recharts, and Gemini AI API integration.

## 🚀 Live Demo

🔗 Add your Vercel deployment link here

Example:

```bash

```

# ✨ Features

* 📊 Interactive fitness analytics dashboard
* 🤖 AI-generated personalized workout plans using Gemini AI
* 🏋️ Workout CRUD operations (Add, Edit, Delete)
* 📈 Calories & workout analytics charts
* 📱 Fully responsive modern UI
* 🌙 Clean dark-themed dashboard design
* ⚡ Fast React + Vite architecture
* 💾 Persistent local storage support

---

# 🧠 AI Functionality

The application integrates Google Gemini AI API to generate personalized workout recommendations based on:

* Fitness goals
* Experience level
* Workout frequency
* Focus areas

Example outputs include:

* Push Pull Legs plans
* Hypertrophy programs
* Fat loss training
* Athletic conditioning routines
* Recovery & nutrition recommendations

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/fitness-tracker-ai.git
```

## Navigate to Client

```bash
cd fitness-tracker-ai/client
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `client` folder:

```env
VITE_GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

# 📦 Build for Production

```bash
npm run build
```
